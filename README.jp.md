# Stardew Valley Tool

[English](README.md) | [简体中文](README.zh.md) | [Português](README.pt.md) | [Español](README.es.md)

Stardew Valley のセーブデータに含まれる一部の情報を確認し、安全に編集するための Windows デスクトップアプリケーションです。Windows の標準セーブデータ保存先を使用し、リリース版の実行ファイルを使用する場合は Python のインストールは不要です。

## ダウンロード

最新の [GitHub Release](https://github.com/fyihang/StardewValleyTool/releases) から `StardewValleySaveManager.exe` をダウンロードしてください。リリース版の実行ファイルを使うために Python をインストールする必要はありません。

## 編集できる項目

- 農場主の名前
- 農場名
- 好きなもの
- 既存の動物の名前（馬がいる場合は馬の名前を含む）
- 選択リストから指定する既存の農場労働者の名前と好きなもの（農場名は引き続き所有者のみが設定できます）

インターフェースは英語、簡体字中国語、ポルトガル語、スペイン語、日本語に対応しています。動物の種類は選択した言語に翻訳されますが、不明な MOD 由来の動物の種類はゲーム内の元の値のまま表示されます。

左下の **About** を使うと、作者、リポジトリ、インストール済みのバージョンを確認でき、それぞれの GitHub ページを開けます。

## セーブデータの保護

起動時に、アプリケーションは次の Windows 標準ディレクトリを検索します。

```text
%appdata%\StardewValley\Saves
```

**セーブデータのフォルダーを選択** を使うと、現在のセッションに限り別のセーブデータのルートを選択できます。この選択は保存されないため、次回起動時には再び標準ディレクトリから開始します。

セーブデータとして認識されるのは、そのフォルダーに `SaveGameInfo` と、フォルダー名と同じ名前のメインセーブファイルの両方がある場合だけです。両ファイルに共通する所有者情報は両方に更新されます。農場労働者と動物は、それらの記録が保存されているメインセーブファイルだけで編集されます。

書き込み前に、アプリケーションは 2 つのファイルをセーブデータのフォルダー内の `.svt-backups/<UTC タイムスタンプ>/` にコピーします。一時 XML を置き換え前に検証し、保存後にも結果を確認します。処理中にエラーが発生した場合は、バックアップから元の 2 つのファイルを復元します。

## 使い方

1. Stardew Valley を完全に終了します。
2. `StardewValleySaveManager.exe` を起動します。
3. 左側の一覧からセーブデータを選択します。
4. 所有者の項目を編集します。農場労働者がいる場合は一覧から選択し、動物の名前をダブルクリックすると名前を変更できます。
5. **変更を保存** を選び、ゲームが終了していることを確認します。

初めて使用する際は、セーブデータのフォルダーを別途コピーし、そのコピーで変更を試してください。ゲーム内で編集後のセーブデータを確認するまで、作成された `.svt-backups` ディレクトリを保持してください。

## ソースコード

ソースコードから実行するには、Python 3.11 以降をインストールし、リポジトリのルートで次を実行します。

```powershell
$env:PYTHONPATH = "src"
python src\__main__.py
```
